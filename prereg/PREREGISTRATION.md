## 2. Transience metric & classification
Frozen constants in `scripts/gate_a_classify.py`: SMOOTH_K=3 rolling mean;
MIN_STEP=100; EMERGED if smoothed conflict ≥0.8; final = mean of last 10% of
evals (≥3); RECOVERED final ≥0.8; DISPLACED final ≤0.6; UNSTABLE if
final-window range >0.2; control-valid = agree final ≥0.8 AND
(agree max − final) <0.15. Class is a FINAL-STATE property:
dip-then-recover (RECOVERED) vs dip-then-stuck (DISPLACED).

## 4. Theory (f*) — frozen metric + ordering prediction
support_ratio(F,C) = (rule_support+1)/(prior_support+1); exact-pattern
counts (`scripts/count_support.py`); gender families use windowed mode
(first " she"/" he" within k=16 tokens of a single-token gendered cue, both
space variants), others exact-bigram. Counts computed, UNREAD.
- Prediction: Spearman ρ(support_ratio, conflict_final) > 0 in every web
  cell; DISPLACED families have lowest web support_ratio.
- Falsifier: any web cell where a bottom-2 support_ratio family is
  RECOVERED while a top-2 family is DISPLACED (both control-valid).
- The critical-threshold (f* contour) claim will be registered as a
  prediction about new Step-5/6 runs after the phase table exists.

## 5. Mechanism — frozen scalar + predictions
S1(ckpt) = cos(mean_girls wte − mean_boys wte, u_she − u_he), names averaged
over both token variants; S2 = ||u_she||/||u_he|| control
(`scripts/mech_gender_scalar.py`).
- M1: web packed final S1 < 0.5×peak, 3/3 seeds; TinyStories packed final
  S1 ≥ 0.5×peak, 3/3 seeds.
- M2: web S1 peak step ≤ behavioral collapse onset (last step with smoothed
  pronoun conflict heldout acc ≥0.5).
